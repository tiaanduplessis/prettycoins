## Install

```sh
$ npm install -g prettycoins
# OR
$ yarn global add prettycoins
```

## Usage

```sh
$ prettycoins
# ...
$ prettycoins --currency=ZAR --limit=10
```

Default currency is `USD` and default limit is `20`.
